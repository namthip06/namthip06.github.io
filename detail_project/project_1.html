Book&Look

Book&Look เป็นโปรเจคที่เกิดจากความฝันที่อยากเปิดร้านขายหนังสือ บวกกับต้องการทำโครงงาน full Stack เพื่อเป็นผลงานรอบ portfolio จึงเกิดเว็บไซต์ขายหนังสือออนไลน์ภายใต้ชื่อ Book&Look ขึ้นมา

โดยเว็บไซต์นี้เขียนขึ้นมาโดยใช้ php เป็นหลัก เพื่อใช้เชื่อมต่อกับฐานข้อมูล mysql ซึ่งจะคอยเก็บข้อมูลต่างๆ เช่น ข้อมูลลูกค้า อยู่ที่ไหน ชื่ออะไร, รายละเอียดของหนังสือ, คำสั่งซื้อของลูกค้า โดยคำสั่งซื้อนั้นจะแบ่งออกเป็น 2 colum หลักๆ ได้แก่ order ซึ่งจะเป็นคำสั่งซื้อหลัก ว่าสั่งซื้อสินค้ากี่ชนิด โดยที่ไม่ขึ้นกับปริมาณการสั่งซื้อ, order_item คือรายละเอียดของคำสั่งซื้อสินค้าแบบชิ้นต่อชิ้น เช่น สั่งซื้อเป็นจำนวนกี่ชิ้น รหัสของหนังสือที่สั่งซื้อมีค่าเท่าไหร่

หมวดหมู่ของหนังสือภายในเว็บไซต์เขียนด้วยภาษา mysql โดยใช้ ORDER BY ในการจัดเรียงข้อมูลตามเงื่อนไขต่างๆ เช่น Best Seller, New Arrivals และใช้ for loop ในการดึงข้อมูล 4 อันแรกออกมา

ระบบ login, sign up เขียนโดยใช้ Sessions ในการส่งข้อมูลผ่านหน้าเว็บ เช่น error และใช้ if ในการเช็คเงื่อนไข ใช้ method post รับค่า

ระบบตะกร้าสินค้า ใช้ array ในการจัดเก็บตัวเลือกสินค้า และใช้ Session ในการส่งข้อมูล จากนั้นเมื่อผู้ใช้ทำการยืนยันรายการสินค้าแล้ว ข้อมูลคำสั่งซื้อก็จะถูกเก็บไว้ใน mysql
