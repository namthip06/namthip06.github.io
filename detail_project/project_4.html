Tarot Day

เป็นอีกหนึ่งโปรเจคที่ใช้ library tkinter ของ python มาสร้าง windows applications โดยสร้างเป็นเกมสุ่มไพ่สามใบ จากนั้นให้ผู้เล่นเล่าเรื่องใดก็ได้ที่เกี่ยวข้องกับไพ่สามใบนั้น โดยโปรแกรมนี้เป็นแค่เครื่องมือสุ่มไพ่เท่านั้น

โดยขั้นแรกคือการสร้าง ckass card ขึ้นมา และเก็บค่าไว้ 2 ค่า คือชื่อการ์ดและที่อยู่ของการ์ด (เพราะการที่จะแสดงผลรูปภาพของ tkinter นั้นจำเป็นต้องอ้างอิงถึงตำแหน่งของรูปภาพภาพในเครื่อง) รวมทั้งสิ้น 15 การ์ด

ขั้นที่สองคือการ์ดสร้าง list การ์ดขึ้นมา จากนั้นเขียนฟังก์ชั่น random card โดยใช้ random.choice(list_card) และตั้งเงื่อนไขด้วย while loop ว่าถ้าการ์ดที่ได้ซ้ำกันให้ทำการสุ่มใหม่อีกครั้ง

ขั้นตอนสุดท้ายคือการสร้างตัวแปรเพื่ออ้างอิงที่อยู่ของการ์ดเข้ากับหน้าต่างแสดงผล => card_tuple[1].location อย่างแรกเพราะการ์ดใช้ฟังก์ชั้นท ramdom card จะคืนค่าออกมาเป็น list จึงจำเป็นต้องกำหนดเลข index เพื่ออ้างอิง อย่างที่สองคือการเรียนใช้ความสามารถของ class card เพื่อให้แสดงผลที่ตังของไฟล์รูปภาพนั้นๆออกม => file=relative_to_assets(card_tuple[1].location)) โดย relative_to_assets เป็นฟังก์ชั่นที่ใช้เข้าถึงโฟลเดอร์ที่เก็บไฟล์รูปภาพของโปรแกรม
