CURRENCY CONVERTOR

WINDOWS APPLICATIONS ที่เขียนโดยใช้ LIBRARY TKINTER ของ PYTHON เป็นเครื่องมือสร้างหน้าตา UI และใช้ API FOREX-PYTHON ในการรับค่าของสกุลเงินในวันนั้น และสถิติในอดีต เพื่อนำมาแสดงผลในหน้าต่างด้านขวา

โดยขั้นแรกคือการตั้งค่า API และรับวันเวลาโดยการใช้ฟังก์ชั่น DATETIME เพราะการใช้งานจำเป็นต้องมีข้อมูลสามอย่าง คือ สกุลเงินต้นทาง ปลายทาง วันเวลา ซึ่งวันเวลาจะแสดงผลอีกสี่ค่าด้วยกัน ได้แก่ เมื่อวาน หนึ่งอาทิตย์ก่อน หนึ่งเดือนก่อน และหนึ่งปีก่อนดังที่แสดงอยู่ฝั่งขวาของหน้าจอ

ขั้นต่อมาคือการสร้างหน้าตา UI ของแอพพลิเคชั่น และสร้างกล่องรับ INPUT เพื่อรับตัวแปรสกุลเงิน และ OUTPUT เพื่อแสดงค่าเงินที่แปลงได้ออกมา โดยใช้ ENTRY ซึ่งสามารถเป้นได้ทั้งกล่องรับค่าและกล่องแสดงผลได้ในตัวเดียว นอกจากนี้ยังตกแต่งด้วยรูปภาพลายเส้นน่ารักๆจาก PhotoImage และสร้างปุ่มต่างๆโดยใช้ button
