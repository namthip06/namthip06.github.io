THAITONE

โครงงานนี้จัดทำขึ้นจากความต้องการประยุกต์ใช้ความรู้ที่เรียนมากับสิ่งรอบตัว และแรงบันดาลใจจากการเห็นเว็บไซต์รวมพาเลตสีของต่างประเทศ ทำให้ข้าพเจ้าเลือกที่จะทำเว็บไซต์รวบรวมข้อมูลสี โดยสีที่เลือกคือสีไทยโทนของไทย

website เขียนโดยใช้ php ในการเชื่อมต่อและดึงข้อมูลจากฐานข้อมูล mysql โดยใช้ for loop ให้ดึงข้อมูลเท่าจำนวนข้อมูลที่มี จากนั้นเขียน php ให้เปลี่ยนสีพื้นหลังของช่องตามรหัสสีที่ดึงมา

นอกจากนี้ยังทำระบบ search ที่รับค่าด้วย method post จากการที่ผู้ใช้กรอกข้อมูลเข้าไปในช่องค้นหา โดยการนำข้อความที่ได้ไปต่อใน code mysql โดยข้อมูลที่อยู่ภายในฐานข้อมูล mysql ได้แก่ ชื่อสี, hex code รวมทั้งสิ้น 333 สีด้วยกัน

ข้อมูลสีทั้งหมดนี้อ้างอิงมาจากเอกสาร THAITONE 2.0 ของ อ.ไพโรจน์ พิทยเมธี
