Line Weather

ใช้ python library requests และ BeautifulSoup ในการเขียน เริ่มจากการดึงข้อมูลจากเว็บไซต์ The Weather Channel จากนั้นจึงใช้ API line notify ในการส่งข้อมูลสภาพอากาศที่ได้มาเข้าไลน์

ขั้นแรกเริ่มจากการติดตั้ง requests, BeautifulSoup และ time รวมถึงตั้งค่า line notify ให้เรียบร้อบก่อนจะเข้าสู่ขั้นต่อไป

ขั้นที่สองคือการเริ่มทำ web scraping กับเว็บไซต์เป้าหมายอย่าง The Weather Channel และใช้ requests ในการดึงข้อมูลภายในหน้านั้นออกมา

ขั้นที่สามคือการใช้คำสั่ง .find เพื่อดึงข้อมูลต่างๆ เช่น อุณหภูมิ สภาพท้องฟ้า โดยการระบุ class หรือ id เช่น => .find('div', class_='CurrentConditions--primary--2DOqs') และถ้าหากข้อมูลมีหลายค่าจะต้องใช้ for loop เพื่อเก็บค่าทั้งหมดลงในตัวแปร list ที่สร้างเอาไว้

ขั้นที่สี่คือการส่งข้อมูลที่ได้ผ่าน line โดยการกำหนดรูปแบบของข้อมูล จากนั้นจึงส่งข้อมูลโดยใช้ฟังก์ชั่น line เป็นอันเสร็จสมบูรณ์
